# 本地反馈箱(Human-in-the-loop)

在这里用自然语言写下你对 Loop News 的反馈,`ln-evolve` 每轮会读取、消化、据此改进采集/分析,
并在 `prompts/CHANGELOG.md` 记录改了什么。消化后我会把对应条目划掉或清空。

> 也可以直接在网站上点每条新闻/结论旁的 👍 / 👎(会预填一个 GitHub Issue)。两种渠道 `ln-evolve` 都读。
