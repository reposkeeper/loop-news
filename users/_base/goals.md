# goals.md — 个人北极星 · base

> **承全局 [GOALS.md](../../GOALS.md)**,把「非显然洞察」这一唯一靶心落到**这一个人**身上。base 版 ≡ 全局北极星(中性、不偏科);分叉后通道 B 只调「对他而言」的侧重,**硬约束一字不改**。

## 个人北极星(单一靶心)
**对这个人而言,最大化「非显然洞察」——产出他从任何单条新闻、甚至从他日常看的主流报道里都得不出的、跨时间/跨域的、可证伪的深层判断,形成他的信息优势。**

衡量每天成功的那一个问题:
> "今天给他这一屏里,有没有几条是——只读任何**单一**来源都得不出、且**言之有据**、**未来可验证**、并且**对他这个人是新的/非显然的**判断?"

## 「对这个人而言非显然」= base 的两条边
- **base(新用户)**:non-obvious 的判据 = 全局判据(需 ≥2 证据、且跨 ≥2 日期**或** ≥2 领域),不预设他的偏好。
- **分叉后**:在他关注的领域做到**产业层纵深**(承全局子目标「领域纵深」),同时——**「对他非显然」不等于「贴他兴趣」**:主流他已看过的,即使贴兴趣也不算洞察;圈内原声、共识缺口、逆共识里他没接触过的,才是给他的信息优势。

## 个人硬约束(承 GOALS,不可为个性化牺牲)
- **不臆造**:非显然 ≠ 凭空联想,也 ≠ 投其所好硬凑。每条洞察必须回链 ≥1 真实 corpus 条目,按 `事实 / 推断 / 预测` 分级标注置信度。
- **宁缺毋滥**:宁可这一屏少一条,也不出「聪明但无据」或「讨好但无据」的结论——**over-reach 是最大风险**,个人进化必须主动压制。
- **不喂气泡**:个性化的下限是**不把他关进回音室/过滤气泡**——始终留 serendipity(由 `rubric.md` 个人克制/创新维钉死)。

## 反目标(个人尺度明确**不**优化)
- 不以「他点得最多」为靶心(那会滑向过滤气泡);点击/采用是**校准「什么对他算洞察」的信号**,不取代北极星。
- 不为贴合度牺牲「不臆造」;不为求新制造噪音。
- 不含系统迭代能力(那是 owner 的全局尺度,不进个人分)。
