# GOALS.md — Loop News 进化北极星

> LOOP 每轮自进化(`ln-evolve`)都朝这个**唯一靶心**收敛。冲突时:北极星 > 子目标 > 顺手优化。
> 跨 agent 通用(Claude / Codex 皆以此为目标函数)。

## 北极星(单一靶心)
**最大化「非显然洞察」—— 产出别人从单条新闻、甚至从主流报道里都得不出的、跨时间 / 跨域的、可证伪的深层判断,形成信息优势。**

Loop News 是一台「情报 → 洞察」引擎,**不是新闻搬运工,也不(主要)为自媒体选题服务**。
衡量每期成功的那一个问题:
> "今天这期里,有没有几条是——只读任何**单一**来源都得不出、且**言之有据**、**未来可验证**的判断?"

## 北极星指标(`ln-synthesize` 计算并写入 `state/metrics.json`,`ln-evolve` 持续追踪,自进化日志可回看)
每期 `data/analysis/<date>.json` 统计:
- **non_obvious** —— 结论中"需 ≥2 证据、且跨 ≥2 日期 **或** ≥2 领域"的条数(**核心指标**)
- **edge** —— `共识缺口` + `矛盾检测` 透镜命中数(主流叙事 ≠ 圈内原声之处 = 信息优势之源)
- **cross_date_links** —— 跨日期关联 / 线索条数
- **falsifiable** —— 带置信度的「预测」条数(前瞻边际)
目标:这几项随轮次**上行**,且**靠质量而非堆数量**。

**运行化 = 四个系统分数(奖惩仪表盘)**:关联度 / 数量 / 分析整合 / 自进化广度 + 综合,每轮 `scripts/score.py` 确定性重算、写 `state/scores.json`(含与上轮 delta)。**agent 每轮都要看分数,优先把最低 / 下滑的那个提上去**;数量分对欠采是**平方级陡峭惩罚**(采集太少即暴跌)。制度与公式见 [prompts/scoring.md](prompts/scoring.md)。

## 子目标(服务北极星的代理,ln-evolve 实际在调)
1. **关联深度↑** —— 跨时间 / 跨域(时间线追踪、二阶效应、跨域模式匹配、跟着钱走)
2. **共识缺口 / 矛盾↑** —— 主流 vs 圈内的分歧,是最高价值信号
3. **原声稀缺度↑** —— 第一手深度发言 > 大路货新闻
4. **信噪比↑** —— 砍水文 / 重复 / 低重要性,把篇幅腾给洞察
5. **个性化(输入而非靶心)** —— 反馈(赞 / 踩 / 采用)用来**校准"什么算有洞察"**;`采用` 是"该洞察确被使用"的旁证,但不取代北极星
6. **领域纵深↑** —— 对用户「关注」的领域,自进化出专属研究方式(KOL/KOC、历年数据、周边产业),长出持续追踪的「📂 专题」(`config/domains.yaml` + `ln-dossier`),把非显然洞察做到产业层

## 采集广度 —— 洞察的前置燃料(奖励维度,与"呈现宁缺毋滥"分层)
洞察是从**料**里萃取的;料太少,再好的方法论也挖不出东西。故**分两层、各有靶**:
- **采集层(高召回,奖励"量")**:广撒网、宁多勿漏,达到 `config/loop.yaml` 的 `collect.volume_target`(≈20/轮)。**欠采(低于 `volume_floor`)= 失职**——`ln-evolve` 要据此补来源 / 加检索角度。用户反馈"新闻太少"就是欠采信号,必须优先修。
- **呈现层(高精度,宁缺毋滥)**:从大池子里**萃取**要闻 / 原声 / 结论,去重、砍水文,以非显然洞察为靶。
- 二者不矛盾:**先有量的池子,才有质的萃取**。`ln-evolve` 同时看 `collected`(量,要够)与北极星四指标(质,要上行);两者都掉才算退步。

## 不可为北极星牺牲的不变量(硬约束)
- **不臆造**:非显然 ≠ 凭空联想。每条洞察必须回链 ≥1 真实 corpus 条目,并按 `事实 / 推断 / 预测` 分级标注置信度。
- **宁缺毋滥**:宁可少出一条,也不出"聪明但无据"的结论 —— **over-reach(过度引申)是本北极星最大的风险,ln-evolve 必须主动压制**。
- 中文呈现 + 深度类保留原文;新闻 / 引文 / 链接真实可回溯;改动可 git revert + CHANGELOG 回滚。

## 反目标(明确**不**优化)
- 不以"覆盖最全 / breaking 最快"为**最终**靶心——它们是采集层的手段与前置(采集层要够量),但最终靶心是洞察。
- 不以"采用率 / 流量"为唯一目标(采用是校准信号)。
- **呈现层**不堆条数充数(**采集层则相反:要够量、欠采即失职**);任何层都不为漂亮指标牺牲"不臆造"。
